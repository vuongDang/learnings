# Group theory 

## Definitions

- __Order of a group__ number of elements in the group
- __Order of an element P in a group__: number of times _k_ it needs to be added to itself such that _kP = 0_
  - if the order of a group is a prime number _N_ then any element of the group is a generator (except the identity element)
  - it also means that all elements of this group have order _N_
- __Lagrange theorem__ the order of any subgroup of a group _G_ must be a modulo of the order of _G_
- __Cyclic group__ if there exists a generator g such that all elements of the group 
  can be obtained via scalar multiplication of g, _kG_
  - in a cyclic group, the order of the generator is the order  of the group
